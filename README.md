# Тестовое задание

## Для запуска проекта:
### `npm install`
### `npm start`

Поочередно запустить в терминале команды:

```bash
npm i
npm start
```

Проект будет запущен по адресу (локально) // Runs the app in the development mode.\

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
